# Express TypeScript Hello World

使用 Express.js 和 TypeScript 构建的简单 Hello World 应用。

## 技术栈

- Express.js
- TypeScript
- nodemon（用于开发环境自动重启）
- pnpm（包管理器）

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发模式运行

```bash
pnpm dev
```

服务器将在 http://localhost:3000 启动。

### 构建生产版本

```bash
pnpm build
```

### 运行生产版本

```bash
pnpm start
```
